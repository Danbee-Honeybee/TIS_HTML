특정 타입에 스타일 주기 206p

1. $("요소선택[속성]") : 선택한 요소중에 지정한 속성이 포함된 요소만 선택함
2. $("요소선택[속성^=값]") : 선택한 요소중에 속성값이 명시한 값으로 시작하는 요소만 선택함
3. $("요소선택[속성$=값]") : 선택한 요소중에 속성값이 명시한 값으로 끝나는 요소만 선택함
4. $("요소선택[속성$=값]") : 선택한 요소중에 속성값이 명시한 값을 포함하는 요소만 선택함
4. $(":type 속성값") : input 요소중 type 속성값이 일치하는 요소만 선택함
